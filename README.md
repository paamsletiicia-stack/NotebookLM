# 📚 NotebookLM - Estudos em C#

## 📖 Sobre o Projeto

Este projeto foi desenvolvido utilizando o **NotebookLM** como ferramenta de apoio aos estudos, com o objetivo de aprofundar conhecimentos na linguagem de programação **C#**.

Através da organização de conteúdos, análise de materiais de referência e aplicação de técnicas de **Engenharia de Prompts**, foi possível tornar o processo de aprendizagem mais estruturado, facilitando a compreensão de conceitos, boas práticas e fundamentos da linguagem.

## 🎯 Objetivo

- Aprimorar os conhecimentos em C#;
- Utilizar Inteligência Artificial como suporte ao aprendizado;
- Organizar informações de forma clara e eficiente;
- Desenvolver habilidades em Engenharia de Prompts para potencializar os estudos.

## 🚀 Tecnologias e Ferramentas Utilizadas

- NotebookLM
- Engenharia de Prompts
- C#
- Inteligência Artificial Generativa

## 💡 Aprendizados

Durante o desenvolvimento deste projeto, foi possível explorar formas mais eficientes de interação com ferramentas de IA, criando prompts estratégicos para:

- Resumir conteúdos técnicos;
- Explicar conceitos complexos de forma simplificada;
- Apoiar revisões e fixação do aprendizado;
- Estruturar materiais de estudo de maneira organizada.

## 📌 Considerações Finais

Este projeto demonstra como a combinação entre **Inteligência Artificial**, **Engenharia de Prompts** e **aprendizado contínuo** pode potencializar os estudos em tecnologia, tornando a jornada de aprendizado mais dinâmica, produtiva e eficiente.
